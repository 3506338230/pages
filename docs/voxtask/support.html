关于本应用

VoxTask 是一款“语音优先”的备忘录工具：按住录音 → 松开识别 → 转文字并保存为任务。语音识别使用字节跳动 ASR（加密传输，直连服务）。

快速开始

  1. 首次运行请允许“麦克风”权限。
  2. 打开 Settings → ASR Configuration：填写 App ID / Token / Cluster。
  3. 可选：Settings → AI Model Configuration：填写大模型 API（用于关键词标题/润色）。
  4. 回到 All 页面，点击“+”开始录音并保存。

常见问题（FAQ）

  • 点击识别后长时间无结果？ 检查网络是否可访问外网；在公司/学校代理网络下 WSS 可能被拦截，建议切换无代理 Wi‑Fi 或蜂窝网络。
  • 识别失败/错误码？ 1001 参数无效（检查 AppID/Token/Cluster）；1012 音频格式无效（使用内置录音，16kHz/16bit/mono）；1020 等待超时（网络抖动或长时间未发送音频，重试）；1005 服务器繁忙（稍后重试）。
  • 如何清除敏感信息？ 开发版可在 Settings → Data & Storage 清除；发行版默认已隐藏。
  • 是否会上传数据到你们自己的服务器？ 不。音频仅加密发送至 ASR 服务以获得转写；转写文本保存在你的设备本地；我们不自建收集服务。

数据与安全

  • ASR/LLM 凭证保存在 iOS Keychain。
  • 仅在你发起识别时使用麦克风；音频不在本地或自有服务器长期存储。
  • 详见《隐私政策》。
