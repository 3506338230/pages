About

VoxTask is a voice‑first memo tool: press and hold to record → release to transcribe → save as a task. Speech recognition uses ByteDance ASR over encrypted transport.

Quick start

  1. Allow microphone access on first launch.
  2. Settings → ASR Configuration: fill App ID / Token / Cluster.
  3. (Optional) Settings → AI Model Configuration: fill your LLM API for keywords/polish.
  4. Back to All, tap “+”, record, release to transcribe, and save.

FAQ

  • No result for a long time? Check network reachability; enterprise proxies may block WSS. Try Wi‑Fi without proxy or cellular.
  • ASR errors? 1001 invalid parameters (check AppID/Token/Cluster); 1012 invalid audio format (use built‑in recorder 16kHz/16‑bit/mono); 1020 timeout (retry); 1005 server busy (retry later).
  • Clear sensitive info? In debug builds: Settings → Data & Storage; release builds hide this section.
  • Do you upload my data to your servers? No. Audio is sent only to ASR for transcription; text stays on your device.
